# Displacement-Force Data Integration Tool

這是一個使用 Streamlit 建構的 Web 應用程式，旨在整合、分析與視覺化多個位移-力 (Displacement-Force) 的測試數據檔案。

此工具能夠讀取多個數據集，根據時間戳進行合併，計算關鍵統計數據，並提供互動式的視覺化圖表。

## 主要功能

- **多檔案上傳**: 支援同時上傳多個 `.csv` 和 `.xlsx` 格式的檔案。
- **數據整合**: 根據共同的 `Time_s` 欄位，將所有來源檔案的數據合併成一個數據集。
- **統計分析**: 在每個時間點上，自動計算位移和力的**平均值 (avg)**、**標準差 (std)**、**最大值 (max)** 和 **最小值 (min)**。
- **互動式視覺化**:
  - 繪製平均的「位移 vs. 力」曲線。
  - 透過帶有標準差誤差線的散佈圖，展示數據的變異程度。
- **數據匯出**: 允許使用者將最終整合並計算完成的數據下載為單一的 CSV 檔案。

## 輸入檔案格式要求

為了讓應用程式能正確讀取，您的輸入檔案 (`.csv` 或 `.xlsx`) 必須遵循以下結構：

1.  應用程式會**自動跳過前兩行**，從第三行開始讀取數值數據。
2.  因此，建議第一行為欄位名稱，第二行為單位。
3.  資料欄位必須依序為 **時間 (Time)**、**位移 (Displacement)**、**力 (Force)**。

**範例結構:**

| 時間 | 位移 | 載荷 |
| :--- | :--- | :--- |
| (s) | (mm) | (kN) |
| 0 | 0 | 0 |
| 0.02 | 0.0001 | 0 |
| 0.04 | 0.0002 | 0.0001 |
| ... | ... | ... |

## 安裝與環境

請先確保您的環境已安裝 Python 3.8+，接著安裝必要的函式庫：

```bash
pip install streamlit pandas plotly openpyxl
```

## 如何執行

1.  在您的終端機 (Terminal) 中，切換到專案所在的目錄。
2.  執行以下指令：
    ```bash
    streamlit run int_app_main.py
    ```
3.  應用程式將會自動在您的網頁瀏覽器中開啟。
